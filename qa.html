Mobicents Sip Servlets Load and Performance Tests

Report of a Mobicents Sip Servlets 1.0 Soak Test on a Jboss Labs server : a current call rate of 450 calls per second over a 74-hour duration for a total of 120,000,000 calls.

Report of a Mobicents Sip Servlets 0.9 Soak Test on a Jboss Labs server : a current call rate of 370 calls per second over a 75-hour duration for a total of 100,000,000 calls.

Report of a Mobicents Sip Servlets 0.7.1 Soak Test on a Jboss Labs server : a current call rate of 350 calls per second over a 80-hour duration for a total of 100,000,000 calls.

Mobicents Sip Servlets version 0.6 can handle load peaks of more than 400 cps. See Report of a Mobicents Sip Servlets Performance Test

The Soak tests displays the results of the latest run of Mobicents Sip servlets load tests.

The Load Test is using a simple call flow :

  • sipP sends INVITE
  • Sip Servlet application sends OK and create a Timer to fire the BYE in 60 seconds
  • sipP sends ACK
  • Sip Servlet application Timer expires and BYE is sent
  • sipP replies OK to BYE
